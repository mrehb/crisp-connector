# Forward to Distributor Plugin

Private Crisp plugin for BigMax Golf support team.

## Features
- Shows customer country and distributor email
- One-click forward to country distributor
- Sends customer notification with distributor contact
- Assigns conversation to Golf Tech Helpdesk

## Usage
1. Open any customer conversation
2. Check distributor info in right sidebar
3. Click "Forward to Distributor" button
4. Customer receives notification with distributor contact
